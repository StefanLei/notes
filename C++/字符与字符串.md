#### 字符串

##### c - style

```cpp
// 相当于是字符数组，结尾有一个 '\0' 空字符
char p[] = {'a','b','c','\0'};
// 输出的是 abc
std::cout << p << std::endl; 


// 把字符数组赋值给字符指针也是可以的
char *b = a;
// 输出的是 abc
std::cout << b << std::endl; 


// 自动添加结束符号在末尾
char c[] = "abc";
// 输出的是 abc
std::cout << c << std::endl; 


// 这样是错误的，因为需要 7 个空间
// char d[6] = "Daniel";
```

##### c++ string

```cpp
// 使用字面值初始化。字面值实际上是，字符数组（结尾有\0)
std::string a = "hello";
// 这样初始化也可以
std::string a("hello");


```
