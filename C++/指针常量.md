#### 指针常量（指针本身是常量，不能修改他的指向）

```c
int a = 1;
int b = 2;


// 这个指针是常量，所以不能修改他的指向
int *const p = &a;

// 这是错误的
// p = &b;

```



#### 常量指针（指向一个常量的指针，不能修改常量的值）

```c
int a = 1;
int b = 2;


// 另一种写法 const int *p = &a;   这两种是等价的，因为常量本身可以用 const int a; 和 int const a;
int const *p = &a;


// 这是错误的，不能修改常量的值
// *p = 123;
```



**记忆方法 const 在谁的后面，就代表谁是常量。 **
